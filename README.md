# Networking Labs - Cisco Packet Tracer

A collection of networking lab projects built in Cisco Packet Tracer (v8.2), covering routing protocols, VPNs, subnetting, and device security.

## Labs

| # | Lab | Concept |
|---|-----|---------|
| 01 | [Inter-VLAN Routing with RIP](01-inter-vlan-with-rip-routing/) | Inter-network routing, RIP, Class A/C addressing |
| 02 | [VPN using GRE over IPsec with OSPF](02-vpn-gre-over-ipsec-with-ospf/) | Site-to-site VPN, GRE tunneling, IPsec encryption, OSPF |
| 03 | [Connecting Different Subnet Classes](03-connecting-different-subnet-classes/) | IP address classes (A & B), inter-class routing, RIP |
| 04 | [Subnetting, Console Access & Static Routing](04-subnetting-console-access-static-routing/) | /26 subnetting, password security, static routing, DCE/DTE |

## How to Use

Each lab folder contains:
- A **README.md** describing the topology, device configurations, and networking concepts
- A **`.pkt` file** - the Cisco Packet Tracer lab file

To open the `.pkt` files, you need [Cisco Packet Tracer](https://www.netacad.com/cisco-packet-tracer) (free for Cisco Networking Academy students).

## Skills Demonstrated

- RIP and OSPF dynamic routing protocols
- Site-to-site VPN (GRE over IPsec) with IKE Phase 1 & 2
- IP addressing and subnetting (Class A/B/C, /26 VLSM)
- Static routing and default routes
- Device security (enable passwords, line passwords, login banners)
- Serial WAN connections (DCE/DTE, clock rate)
- Switch fundamentals (Cisco 2950T, 2960, IE-2000)
